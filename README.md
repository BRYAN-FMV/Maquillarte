# Maquillarte

Sistema de gestión de inventario y ventas.

Estructura básica creada con Vite + React. Contiene servicios que interactúan con Firebase Firestore para ventas, inventario, proveedores y gastos.

Instrucciones rápidas:

- Instalar dependencias: `npm install`
- Ejecutar en desarrollo: `npm run dev`
- Construir producción: `npm run build`

Ver la documentación interna en la carpeta `src/`.
